Hardware and software requirements
- Devices with Android 2.3 and above
- Android Developer Tools v22
- Android SDK v4.3 (API level 18)
- A published layer or a pro campaign in your Layar account. For getting started please see http://devsupport.layar.com/forums/20604191-getting-started or http://devsupport.layar.com/forums/20603886-getting-started

Changelog:

- November 25th - SDK 8.4.3
Fixed Nexus 5X camera orientation issue.
Added support for Android 6.0, including runtime permissions (enabled if you set targetSdk to 23).
The SDK is now distributed only as an .aar library. Support for the Android Library project as used
in Eclipse has been deprecated. If you are still using Eclipse, we recommend switching to Android
Studio as Eclipse support has been officially deprecated by Google.

- July 29th 2015 - SDK 8.4.2
Various bug fixes and improvements

- April 2nd 2015 - SDK 8.4.1
Fixed crash on Galaxy S5 with Android 5.0

- March 3rd 2015 - SDK 8.4

Fully customisable UI for your scan view
Added support to change the view-angle of the pop-out view (2d or 3d view)
Added support to hide the trigger image in the pop-out view
Added support to automatically re-attach the content back to the trigger image after it was opened in the pop-out view 
Added support to tak and share a screenshot 

Added support to loop AR videos (only available using the Layar API)
Added support to hide the audio player (only available using the Layar API)
Added support to hide the loading indicators (only available using the Layar API)
Added support to hide the replay button for AR videos (only available using the Layar API)

Various bug fixes and improvements


- November 3rd 2014 - SDK 8.3.1
Scan issues on Galaxy S4 Mini and some LG devices fixed (autofocus issue)
Fixed scrolling issue in WebPanels on Android 4.4
Memory leak fix

- May 13th 2014 - SDK 8.2
Fix for transparent animated GIF
Fix for buttons not shown in front or back in regards to other buttons
Various other bug fixes and improvements

- March 11th 2014 - SDK 8.1
The Layar SDK has been supplied with pop-out and additional APIs to alter the behaviour for this feature
Various bug fixes and improvements


- October 24th 2013 - SDK 8.0
Support for Geo layers is added. Using Vision layers or Geo layers inside the SDK is subject to you having purchased the appropriate key to unlock that class of content.
A lot of new callback methods added allowing easier customization.
Multiple bug-fixes and performance improvements.

- March 15th 2013 - SDK 7.2.0

Fixed crash to improve stability.
