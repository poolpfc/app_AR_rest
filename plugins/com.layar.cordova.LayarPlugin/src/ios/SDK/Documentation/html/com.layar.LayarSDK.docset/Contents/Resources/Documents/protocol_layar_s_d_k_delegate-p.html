Detailed Description

The LayarSDKDelegate provides you with several callback methods to inform you and allow you to override some activities.

You get information about which layer has been loaded and when a certain reference image is tracked etc.

All callbacks are optional, so you only need to implement the ones you're interested in.

Many callbacks pass Layer and POI objects in form of dictionaries that contain extended information about the campaign/layer and POI for which the callback is made.

These dictionaries always use strings (NSString*) as keys.

The following structure is guaranteed for POI objects: 

            {
                "id"        : (NSString*) The unique id which identifies a POI in the getPOIs response.
                "object"    : (NSDictionary*)
                    {
                        "url"           : (NSString*) The URL of the object to be displayed.
                        "contentType"   : (NSString*) The content type of the object.
                    }
            }

For more information on POI objects see 'hotspots' documentation in the GetPOIs API doc (https://www.layar.com/documentation/browser/api/getpois-response/hotspots/). Note that not all fields specified in the GetPOIs API are available in POI object.

The following structure is guaranteed for Layer objects: 

            {
                "name"      : (NSString*) The name of the layer
                "title"     : (NSString*) The title of the layer
                "icon"      : (NSString*) The URL of the layer's icon
            }

Internally campaigns are related to layers so the layer name is your campaign's name and the layer title is your campaign's title.

willSendPOIRequestWithParameters: (NSDictionary *)  parameters 
optional

Called when the layar sdk will make a request for pois. This callback enables the user of the SDK to customize the parameters being sent with the poi request. The entries in the returned dictionary will override the entries in the parameters before being sent to the server. New entries will be added. Note that in this manner a parameter cannot be removed. It can either be modified or added.

Parameters
layarSdkLayarSDK* Reference to the LayarSDK object
parametersNSDictionary* The parameters being sent with the poi request.
Returns
NSDictionary* containing keys for entries that need to be modified or added to parameters with their value.
